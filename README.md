# Uttej Moturu — Portfolio Website

A professional, single-page portfolio website showcasing my background as a Software Engineer with experience in full-stack development, AI/computer vision, and research-led engineering projects.

## 🚀 Project Overview

This repository contains my personal portfolio website, built as a fast and responsive static web app.  
It highlights my:

- Technical skills
- Resume and education
- Research and development work
- Achievements and certifications
- Project portfolio
- Contact and social links

## ✨ Features & Highlights

- Clean, modern UI with responsive layout
- Multi-section navigation (About, Resume, Portfolio, Contact)
- Downloadable resume (PDF)
- Certifications and achievements showcase
- Skills overview across software, AI/ML, and cloud
- Direct contact options (email, phone, location, social profiles)

## 🛠 Tech Stack

- **HTML5** — structure and semantic content
- **CSS3** — styling, layout, and visual design
- **JavaScript (Vanilla)** — interactive behavior
- **Ionicons** — icon library
- **Google Fonts (Poppins)** — typography

## ⚙️ Installation & Setup

### Option 1: Run locally (quickest)
1. Clone the repository:
   ```bash
   git clone https://github.com/MOTURUUTTEJ/My_Portfolio.git
   ```
2. Move into the project folder:
   ```bash
   cd My_Portfolio
   ```
3. Open `index.html` directly in your browser.

### Option 2: Serve with a local dev server
If you prefer live reload or serving over localhost:

```bash
python -m http.server 8080
```

Then open: `http://localhost:8080`

## ▶️ How to Use

1. Open the site locally or via the live demo.
2. Use the top navigation to explore:
   - About
   - Resume
   - Portfolio
   - Contact
3. Click **Download CV** in the Resume section to get my latest resume.
4. Use the Contact section/social icons to connect with me directly.
